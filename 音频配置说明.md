# 音频录制配置说明

## 问题：录制的视频没有声音

## 解决步骤

### 1. 查找可用的音频设备

运行设备列表脚本：

```bash
python list_audio_devices.py
```

会显示类似这样的输出：

```
[dshow @ 000002b8] "麦克风 (Realtek High Definition Audio)"
[dshow @ 000002b8] "立体声混音 (Realtek High Definition Audio)"
[dshow @ 000002b8] "扬声器 (Realtek High Definition Audio)"
```

### 2. 选择音频设备

**推荐使用：立体声混音（Stereo Mix）**
- ✅ 可以录制系统声音（课程音频）
- ✅ 不需要麦克风

**备选方案：**
- 如果没有立体声混音，使用麦克风
- 但会录制环境噪音

### 3. 修改录制脚本

编辑 `auto_course_recorder.py` 第233行：

```python
'-i', 'audio=你的音频设备名称',
```

**示例1：立体声混音**
```python
'-i', 'audio=立体声混音 (Realtek High Definition Audio)',
```

**示例2：麦克风**
```python
'-i', 'audio=麦克风 (Realtek High Definition Audio)',
```

### 4. 启用立体声混音（如果没有）

**Windows 10/11：**

1. 右键点击任务栏的**音量图标**
2. 选择**"声音设置"**或**"打开声音设置"**
3. 滚动到**"更多声音设置"**或**"声音控制面板"**
4. 点击**"录制"**选项卡
5. 右键点击空白区域，勾选**"显示已禁用的设备"**
6. 找到**"立体声混音"**
7. 右键点击，选择**"启用"**
8. 右键点击，选择**"设为默认设备"**（可选）

### 5. 测试音频录制

修改完成后，重新运行录制脚本：

```bash
python auto_course_recorder.py
```

录制一小段视频，检查是否有声音。

## 常见问题

**Q: 没有立体声混音选项？**
A: 某些声卡驱动不提供此功能。可以：
   - 更新声卡驱动
   - 使用虚拟音频线软件（如 VB-Cable）
   - 直接使用麦克风录制（会有环境音）

**Q: 录制了但没有声音？**
A: 检查：
   - 音频设备名称是否正确
   - 立体声混音是否启用
   - 系统音量是否静音

**Q: 如何只录系统声音不录麦克风？**
A: 使用立体声混音，并确保麦克风没有被选为录制设备。

## 替代方案：使用虚拟音频线

如果以上方法都不行，可以安装虚拟音频设备：

1. **下载 VB-Cable**
   - 官网：https://vb-audio.com/Cable/
   - 免费开源

2. **安装 VB-Cable**
   - 下载后解压
   - 运行安装程序
   - 重启电脑

3. **设置音频输出**
   - 系统声音输出 → VB-Cable Input
   - 录制设备 → VB-Cable Output

4. **修改脚本**
   ```python
   '-i', 'audio=CABLE Output (VB-Audio Virtual Cable)',
   ```
